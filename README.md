# Sistema Tebas - Front-End

### Execução

Crie um arquivo .env utilizando .envexample com o exemplo.
Você pode inicializar a aplicação utilizando docker-compose.

`docker-compose up -d`

O acesso é pelo endereço: http://localhost:3000
